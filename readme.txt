Smart Contact Buttons tạo ra các nút gọi điện và liên hệ ngay hiển thị trên màn hình, giúp khách hàng dễ dàng tương tác và liên hệ tới chủ nhân website. 

== Description ==
Với Smart Contact Buttons, bạn sẽ được tạo các nút như: Gọi điện, Zalo, Messenger, Google Maps với giao diện bắt mắt, tính thẩm mỹ cao.


Bạn có thể chỉnh sửa các thông tin này một cách dễ dàng trong Admin.
- Chỉnh sửa số điện thoại và chữ hiển thị
- Chỉnh sửa Số điện thoại Zalo
- Chỉnh sửa ID của fanpage giúp khách hàng nhắn tin bằng Messenger
- Chỉnh sửa link Google Maps giúp khách hàng biết vị trí trên bản đồ

== Frequently Asked Questions ==

Q: Smart Contact Buttons phù hợp với trang web nào?
A: Smart Contact Buttons thích hợp với các trang web bán hàng, giới thiệu sản phẩm, landingpage

== Cài đặt ==
1. Upload thư mục \'smart-contact-buttons\' vào wp-content/plugins
1. Kích hoạt plugin trong khu vực Plugins -> Installed Plugins
1. Vào mục Smart Contact Buttons bên trái Menu để cài đặt.
